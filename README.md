# 🚀 React Authentication System

A **secure and dynamic authentication system** built using **React, Node.js, Express, and MongoDB**. This system handles user login, registration, and dashboard access with JWT authentication. It also dynamically displays the logged-in user's initial in a profile icon in the corner.

---

## 📚 Features

- 🔒 User Registration & Login
- 🔐 JWT-based Authentication
- 📄 Protected Routes with Role-Based Access
- 🟢 Dynamic Profile Icon with User Initial
- 🔄 Logout & Session Management
- 🎨 Beautiful UI with Responsive Design

```bash
